# RF-ONE Core Principles

**Document Version:** 5.0
**Status:** Approved (Core 1.0 Freeze)
**Module:** Core

# Purpose

This document defines the immutable principles governing the RF-ONE Core.

Every RF-ONE concept belongs to one and only one layer:

- Core
- Domains
- Environment

The Core models universal reality.
Domains model business knowledge.
Environment models contextual knowledge.

## Principles

1. Core before technology.
2. Only universal concepts belong to the Core.
3. Reality before software.
4. Every concept with identity is an Entity.
5. Identity is immutable.
6. Relationships are independent.
7. Goals define outcomes only.
8. Processes define how goals are achieved.
9. The system chooses the executor according to available capabilities and resources.
10. Business knowledge is expressed through Processes.
11. Composition before specialization.
12. Single Source of Truth.
13. Extensibility without breaking existing concepts.
14. AI reasons about the Core but is not part of it.
15. Facts are persisted; conclusions are inferred whenever possible.

# Guiding Principle

The Core is the permanent intellectual asset of RF-ONE.
Technologies, Domains and AI evolve.
The Core remains stable because it models reality itself.
